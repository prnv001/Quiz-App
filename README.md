# 🧠 Pro Quiz App

A fully-featured Quiz Application built with HTML, CSS, and JavaScript, designed to simulate a real-world quiz experience. Great for learning core web development concepts and demonstrating frontend skills during interviews.

---

## 🔥 Features

- ✅ Start screen with animated transitions
- ✅ Timed quiz with visual progress bar
- ✅ Randomized questions and shuffled answer options
- ✅ Instant feedback on answer selection
- ✅ Score and total time tracking
- ✅ Save high scores in browser using `localStorage`
- ✅ Leaderboard with name, score, and time
- ✅ Responsive, mobile-friendly layout

---
